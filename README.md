# OCR Demo

## Installation

1. macOS: `brew install tesseract tesseract-lang`\
Ubuntu: `sudo apt install tesseract-ocr tesseract-ocr-lav`
2. `pip install pipenv`
3. `pipenv sync`
4. `cp .env.example .env`

## Usage

1. `python main.py`
2. see results in the folder specified by `OUTPUT_DIR` in `.env`
